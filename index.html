🎠 Swiper Carousel Web Component Demo

Basic Carousel

A simple carousel with navigation arrows and pagination dots. Features one slide at a time.
Keyboard Navigation: Use arrow keys (← →) to navigate when focused on the carousel.

Multiple Slides View

Display multiple slides at once with responsive breakpoints.
Responsive: Resize your browser to see different numbers of slides (1 on mobile, 2 on small screens, 3 on tablets, 4 on desktop).

Loop Mode

Infinite loop carousel that wraps around from the last slide to the first.
Loop Mode: Navigate past the last slide to smoothly wrap back to the first, or go backwards from the first to reach the last. Perfect for continuous browsing.

Autoplay

Automatically advance slides with configurable delay. Includes pause on hover and interaction controls.
Autoplay: This carousel automatically advances every 2.5 seconds. Hover over it to pause, or use the controls above to start/stop autoplay manually.

Fade Effect

Smooth crossfade transition between slides instead of sliding. Perfect for image galleries and hero sections.
Fade Effect: Uses crossfade transitions instead of sliding. Works great with autoplay for a smooth, continuous experience.

Custom Content

You can put any web component or HTML content in the carousel.
No Navigation: This carousel has navigation arrows disabled. Use pagination dots or keyboard to navigate.
